Frequently Asked Questions

Are my PDF files uploaded to any server?

No! All PDF processing happens locally in your browser. Your files never leave your device.

How does Image to PDF work?

Select one or more images, choose page settings, then open print layout and use browser Print → Save as PDF. This keeps conversion local and fast.

Can I preview PDF files?

Yes. PDF Viewer & Inspector shows an embedded preview, key file metadata, and quick actions like open tab, print, and metadata copy.

What formats are supported for images?

All common image formats: JPEG, PNG, WebP, GIF, BMP.
